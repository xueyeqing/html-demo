# flex 布局

这是看完阮一峰老师的 *Flex布局教程* 后写的一些demo。

- `base.html` 是基本语法的使用。
- `grid-layout.html` 是网格布局，但是没有做自适应。
- `holy-grail-layout.html` 是圣杯布局。
